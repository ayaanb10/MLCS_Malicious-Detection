# Phishing Detection using Machine Learning

This project implements a machine learning model to detect phishing URLs and emails using feature extraction and classification techniques.

## Features
- URL-based feature extraction (length, symbols, HTTPS, etc.)
- Email-based feature analysis (subject, keywords, content)
- Random Forest classification model
- Prediction of phishing vs legitimate inputs

## Dataset
- Custom dataset (URL + Email)
- Kaggle phishing websites dataset

## Implementation
- Data preprocessing and cleaning
- Feature engineering
- Model training and evaluation
- Prediction on new inputs

## Results
The model achieves effective classification performance with high accuracy and low false positive rates.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
